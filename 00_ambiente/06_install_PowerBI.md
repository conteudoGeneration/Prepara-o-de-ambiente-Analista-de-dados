<h1>Guia de Instalação do Power BI Desktop</h1>

1. Acesse o site oficial da Microsoft:

👉 https://powerbi.microsoft.com/pt-br/desktop/

Clique em **Baixar gratuitamente** para fazer o download do **Power BI Desktop**.

<br/>

| ⚠️ | **ATENÇÃO:** O Power BI Desktop está disponível apenas para Windows. Para macOS utilize máquina virtual ou navegador (Power BI Service). |
|----|------------------------------------------------------------|

---

# 🪟 Instalação no Windows

2. Após efetuar o download, execute o instalador `.exe`.

3. Caso apareça a mensagem de segurança do Windows, clique em:

<div align="center">
<b>Executar</b>
</div>

4. Na tela do instalador, clique em:

```text
Next
```

5. Aceite os termos:

```text
I accept the license terms
```

6. Clique em:

```text
Install
```

7. Aguarde a conclusão da instalação.

8. Clique em:

```text
Finish
```

---

## Abrindo o Power BI

9. Pressione:

```text
Windows
```

10. Pesquise por:

```text
Power BI Desktop
```

11. Clique para abrir o aplicativo.

---

## Verificando funcionamento

12. Ao abrir, você verá a tela inicial com opções:

- Obter dados
- Transformar dados
- Relatórios
- Modelagem

Se essa tela abrir corretamente, a instalação foi concluída com sucesso.

---

# 🍎 Instalação no macOS (Alternativas)

O Power BI Desktop **não possui versão nativa para macOS**.

Você pode utilizar:

## Opção 1 — Navegador (recomendado)

1. Acesse:
https://app.powerbi.com

2. Faça login com conta Microsoft.

3. Utilize o **Power BI Service (online)** diretamente no navegador.

---

## Opção 2 — Máquina virtual Windows

Instale o Windows utilizando:

- Parallels
- VirtualBox
- Boot Camp (Mac Intel)

Depois instale o Power BI normalmente dentro do Windows.

---

# Testando o primeiro relatório

13. Clique em:

```text
Obter dados → Excel ou CSV
```

14. Selecione um arquivo de exemplo.

15. Clique em:

```text
Carregar
```

16. Arraste campos para o relatório para gerar seu primeiro gráfico.

---

# Atualização do Power BI

Para atualizar:

- Abra o Power BI
- Vá em **Arquivo → Opções → Atualizações**
- Ou baixe a nova versão no site oficial

---

# Resumo

| Sistema | Instalação |
|---------|-----------|
| Windows | Instalar Power BI Desktop (.exe) |
| macOS | Usar navegador (app.powerbi.com) ou VM |
| Web | Acesso online sem instalação |
